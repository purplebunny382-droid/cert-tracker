Connect Google Drive
To sync with Google Drive you need a free API key. This is a one-time setup that takes about 3 minutes.
  1. Go to console.cloud.google.com and sign in with your Google account
  2. Create a new project (click "New Project", give it any name)
  3. Go to APIs & Services → Library and enable Google Drive API
  4. Go to APIs & Services → Credentials → Create Credentials → API Key
  5. Also create an OAuth 2.0 Client ID (type: Web application) — add http://localhost as an authorized origin
  6. Paste your Client ID below and click Connect
Your data is saved in a file called cert_tracker_data.json in your Google Drive. Only you can access it.
